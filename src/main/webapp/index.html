Sobre o projeto

Esta é uma aplicação Java Web desenvolvida na arquitetura MVC e é composta das funções Logar, Armazenar informações do usuário em cookies, Listar Consultas marcadas, Criar Consultas, Editar Consultas , Excluir Consultas. O projeto tem o objetivo de demostrar habilidadedes nas tecnologias, JSP, Servlets, JDBC, HTML 5, CSS 3, javascript. Os padrões de projeto utilizados na aplicação são: MVC, Maven e DAO; É um sistema em nivel iniciante no desenvolvimento java para web.
